1. Allgemeines

Diese App dient der persönlichen Verwaltung einer Büchersammlung. Die Nutzung der App ist freiwillig. Im Folgenden wird erläutert, welche Daten erhoben werden und wie sie verwendet werden.

2. Microsoft OneDrive / Microsoft-Konto

Die App ermöglicht eine optionale Anmeldung mit einem Microsoft-Konto, um Buchdaten (Metadaten wie Titel, Autor, Datum; Cover-Bilder) in OneDrive zu sichern und wiederherzustellen. Die Authentifizierung erfolgt über das offizielle OAuth 2.0-Verfahren von Microsoft. Zugangsdaten (Passwörter) werden von der App zu keinem Zeitpunkt gespeichert oder verarbeitet.

Durch die Nutzung von Microsoft OneDrive gelten die Datenschutzbestimmungen von Microsoft: https://privacy.microsoft.com/de-de/privacystatement

Das OAuth-Zugriffstoken wird verschlüsselt auf dem Gerät gespeichert (Android Keystore) und ausschließlich zur Kommunikation mit der Microsoft Graph API verwendet.

3. Kamera

Die App greift auf die Kamera des Geräts zu, um ISBN-Barcodes von Büchern zu scannen. Kameraaufnahmen werden nicht gespeichert, nicht übertragen und nicht an Dritte weitergegeben. Die Kamera wird nur während des Scan-Vorgangs aktiv genutzt und danach sofort freigegeben.

4. Lokale Datenspeicherung

Buchdaten werden lokal in einer SQLite-Datenbank auf dem Gerät gespeichert. Diese Daten verlassen das Gerät nur, wenn der Nutzer aktiv eine OneDrive-Synchronisierung auslöst.

5. Keine Weitergabe an Dritte

Es werden keine personenbezogenen Daten an Dritte weitergegeben, verkauft oder für Werbezwecke genutzt. Die App enthält keine Werbemodule, keine Analytics-Dienste und kein Tracking.

6. Buchmetadaten (Open Library)

Beim Scannen eines ISBN-Barcodes wird eine Anfrage an den öffentlichen Buchkatalog Open Library (openlibrary.org) gestellt, um Titel, Autor und Cover abzurufen. Dabei wird die ISBN-Nummer übertragen. Es werden keine personenbezogenen Daten gespeichert oder an Open Library weitergegeben.

7. Rechte der Nutzer

Nutzer können jederzeit alle lokal gespeicherten Daten löschen, indem sie die App deinstallieren oder die App-Daten im Betriebssystem zurücksetzen. OneDrive-Daten können direkt über das Microsoft-Konto verwaltet und gelöscht werden.
